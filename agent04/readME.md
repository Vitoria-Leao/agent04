# agentTaskManager
Este repositório contém o código em Python utilizado para criar um agente de IA que cria, adiciona e atualiza tarefas no Trello. O projeto foi criado durante o Bootcamp da DIO: CI&T - Do Prompt ao Agente.

## Passo a passo - VS Code
1. Mudar o diretório de trabalho para agent04:
    cd agent04
2. Criar um ambiente virtual:
    python -m venv lab-dio
3. Ativar o ambiente:
    ./lab-dio/Scripts/Activate.ps1
4. Instalar as bibliotecas necessárias:
    pip install -r requirements.txt
5. Criar um arquivo .env com sua chave, segredo e token do Trello.
6. Rodar o agente com adk web:
    adk web
7. Agora é só conversar com o agente para admnistrar suas tarefas.
